# 리액트 네이티브 메모 앱

이 앱은 리액트 네이티브를 사용하여 만든 간단한 메모 어플리케이션입니다. 사용자는 메모를 생성, 편집, 삭제할 수 있으며, 데이터는 AsyncStorage에 저장되어 영구성을 제공합니다.

## 기능

- 제목과 설명이 있는 새 메모 추가
- 그리드 레이아웃에서 모든 메모 보기
- 제목으로 메모 검색
- 기존 메모 업데이트
- 메모 삭제
- 메모는 생성/업데이트 날짜별로 정렬됨
- AsyncStorage를 사용한 메모 데이터 유지
